# Go Hotwire

A sandbox for my attempt at combining:

- [https://github.com/a-h/templ](templ)
- [https://hotwire.dev/](hotwire.dev)

## Prerequisites

- golang
- [https://github.com/a-h/templ](templ)

## Running

This command will compile all templ files then run the server:

`templ generate && go run .`
